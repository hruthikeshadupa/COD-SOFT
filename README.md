## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Git

### Installation
1. Clone the repository:
```bash
git clone https://github.com/hruthikeshadupa/COD-SOFT.git
```

2. Navigate to the project directory:
```bash
cd COD-SOFT
```

3. Compile the Java files:
```bash
javac *.java
```

## Usage
To run any Java program in this repository:
```bash
java ClassName
```

Replace `ClassName` with the specific class you want to execute.
